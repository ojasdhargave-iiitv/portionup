# Security Policy

## Supported Versions

Only the latest version of PortionUp is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Please do not open a public issue if you discover a security vulnerability. Instead, contact the maintainers directly. We will investigate the issue and work on a fix as soon as possible. We strive to acknowledge reports within 48 hours and will keep you informed of our progress.
